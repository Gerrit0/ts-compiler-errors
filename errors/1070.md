When defining a type, no modifiers (except `readonly`) can be applied to its type members.

```ts
interface A {
  async a(): Promise<string>
  public b(): string
  private c(): string
  protected d(): string
}
```

To fix the error, remove the modifiers.

```ts
interface A {
  a(): Promise<string>
  b(): string
  c(): string
  d(): string
}
```
