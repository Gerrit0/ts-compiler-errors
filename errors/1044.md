Most modifiers cannot be applied to members in a namespace. If you try to add them, you will get an error.

```ts
namespace A {
  static const a = 1
  private const b = 2
  protected const c = 3
  private const d = 4
}
```

To fix the errors, remove the unnecessary modifiers.

```ts
namespace A {
  const a = 1
  const b = 2
  const c = 3
  const d = 4
}
```
